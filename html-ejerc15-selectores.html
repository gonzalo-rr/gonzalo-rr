Tipos de selectores que entran en examen

Clase (class="")

Este selector sirve para asignar un estilo a cada etiqueta a la que le asignes una clase.

Ejemplo en código: "<p class="claseejemplo">Párrafo con clase.<em class="claseejemplo"> Aqui también hay clase.<em></p>"

Ejemplo práctico

Párrafo con clase. Aqui también hay clase.

Aqui hay muchísima clase.

Identificador (id="")

Este selector sirve para asignar un estilo a una etiqueta única y específica a la que le quieras asignar un estilo único.

Ejemplo en código: "<p id="estiloejemplo">Texto</p>" "</style><#estiloejemplo{}"

Todos los elementos (*)

Este selector se usa al principio de los estilos y afecta a todas las etiquetas

Para demostrar el efecto cambiaré el estilo de esta sección, se usa así: "<style>*p{padding: 40px; font-size: large;}</style>

Varios elementos ( p, em, h1, h2)

Este tipo de selector sirve para ponerle un mismo estilo a varias etiquetas.

Para su utilización solo hace falta usar "<style>h4,h5,h6{background-color: rebeccapurple;}</style>"

Ejemplo práctico: Pondré el mismo estilo a los h4, h5 y h6.

Elementos contenidos en otros elementos (ol li)

Este selector se usa para dos etiquetas cuando quieres cambiar solo un elemento contenido en otro específico.

Ejemplo en código: "<style>ol il {color:red;}</style>"

Ejemplo práctico:

Sobrevuleo (p:hover)

Este selector cambia el estilo de un texto cuando se pasa por encima el ratón.

Ejemplo en código: "<style>p:hover{color: red; font-family: 'Franklin Gothic Medium', 'Arial Narrow', Arial, sans-serif; font-size: 120%}</style>
